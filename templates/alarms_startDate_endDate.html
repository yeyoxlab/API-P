of alarm, meter alarm and system alarm:
An alert system has the following structure:

{ 
	"TYPE_ALARM":"DCU",
    "CODE_ALARM":17,
    "TIMESTAMP":"2016-01-20 15:34:20"
},

FIELD MEANING TYPE VALUES
TYPE_ALARM indicates the type of alarm STRING
CODE_ALARM Unique code for identifying alert INTEGER > 0
TIMESTAMP Timestamp of alert with Mysql format STRING

Note: The types of alarms are: meter or DCU

{
